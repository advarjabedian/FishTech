Overview

This page helps you monitor daily inspection compliance across all your facilities.

Status Indicators
  • Complete - All SOPs completed for this shift
  • In Progress - Inspection started but not finished
  • Verified - Inspection has been verified by supervisor
Filters
  • Incomplete Days - Shows days with unfinished inspections
  • Unverified Logs - Shows completed but unverified inspections
  • All Days - Shows all inspection days
